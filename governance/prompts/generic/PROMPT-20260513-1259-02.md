# Metaprompt: Bidirectional Drag-and-Drop

**Date**: 2026-05-13 12:59
**Source**: User Request
**Context**: Admin Studio UX Refinement

## Instruction
"Esto no permite arrastrar desde el centro hacia el listado de campos para eliminar un camppo"

## Objective
Allow users to remove fields from the canvas by dragging them back to the palette.

## Implementation Result
- Implemented `handleDragOver` for palette highlight.
- Updated `handleDragEnd` to handle canvas-to-palette moves.
- Added visual feedback (trash icon) in `DroppablePalette`.
